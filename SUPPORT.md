# Support

The Yalla SDK is a Yalla-opinionated Kotlin Multiplatform ride-hailing SDK. It is maintained primarily to serve the YallaClient app; external consumers are welcome but not actively solicited.

## Questions

- **Bug reports**: open a GitHub issue using the Bug Report template.
- **Feature requests**: open a GitHub issue using the Feature Request template. Honest stance: most feature requests are closed as out-of-scope unless they align with YallaClient's roadmap. Open source does not mean open roadmap.
- **Documentation issues**: open a GitHub issue using the Docs template.
- **Security issues**: see [`SECURITY.md`](SECURITY.md) — do not open a public issue for security.

## Response Expectations

This is a best-effort project. No contractual support. Response times vary. The maintainer is a single engineer (`@isloms`).
