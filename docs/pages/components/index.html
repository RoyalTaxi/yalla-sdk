Status & roadmap

Catalog v0.1 — static. This page is hand-curated from the SDK source tree and kept in sync at release time. Every entry is real and shipped in the current alpha; for full parameter lists, slot APIs, and modifier defaults, follow the link to /api/.
Coming next: live catalog. A Storytale-powered Compose Multiplatform/wasm gallery is the planned next step — it will render every component live with controls for states (selected, loading, dark, RTL) under /components/. Blockers tracked in :component-catalog include Kotlin 2.3 + KGP wasmJs / detekt compiler-embeddable alignment, and a wasmJs publication for the connectivity dependency consumed by :composites.
For new services (e.g. intercity). Before adding a screen, check this page for the right primitive/composite to reuse. Promote a feature-private component into the SDK only after it ships in two services and the design has stabilized.
