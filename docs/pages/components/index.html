Maps & media

Provider-agnostic platform integrations from :maps and :media. The maps API abstracts over Google Maps and MapLibre with runtime switching; media wraps native camera, gallery, and image pipelines.

Fully interactive map composable — gestures, overlays, camera control. Use for the home/order map.

Lightweight non-interactive map for previews and list rows. Cheaper than ExtendedMap.

Fully static map (no gestures, no live tiles) — for receipts, history rows, and snapshots.

Status & roadmap

Catalog v0.1 — static. This page is hand-curated from the SDK source tree and kept in sync at release time. Every entry is real and shipped in the current alpha; for full parameter lists, slot APIs, and modifier defaults, follow the link to /api/.
Coming next: live catalog (Phase 2). A Storytale-powered Compose Multiplatform/wasm gallery will replace this page at the same URL — rendering every component live with controls for states (selected, loading, dark, RTL). Three known blockers gate the work: (1) Kotlin 2.3 + KGP wasmJs vs detekt/dokka's transitive kotlin-compiler-embeddable on the buildscript classpath; (2) no wasmJs publication for dev.jordond.connectivity:connectivity-device, consumed by :composites's DeviceConnectivityState from commonMain; (3) Storytale itself is alpha and its API may shift before 0.1.
For new services (e.g. intercity). Before adding a screen, check this page for the right primitive/composite to reuse. Promote a feature-private component into the SDK only after it ships in two services and the design has stabilized.
